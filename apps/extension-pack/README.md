# Android DevKit Pack

A complete Android development environment for VS Code. Install this extension pack to get everything you need for Android app development.

## Included Extensions

| Extension | Description |
|-----------|-------------|
| **[Android DevKit](https://marketplace.visualstudio.com/items?itemName=pavi2410.android-devkit)** | Device manager, Logcat viewer, Emulator management, SDK manager |
| **[Language Support for Java](https://marketplace.visualstudio.com/items?itemName=redhat.java)** | Java language support via Eclipse JDT |
| **[Gradle for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-gradle)** | Gradle build tool integration |
| **[Kotlin](https://marketplace.visualstudio.com/items?itemName=fwcd.kotlin)** | Kotlin language support |
| **[XML](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-xml)** | XML language support for layouts and resources |

## Why Use This Pack?

- **One-click setup** - Get a complete Android development environment instantly
- **Curated selection** - Each extension is chosen for quality and compatibility
- **Lightweight** - Only installs what you need, no bloat
- **Always up-to-date** - Extensions update independently

## Requirements

- Android SDK installed (`ANDROID_HOME` or `ANDROID_SDK_ROOT` configured)
- JDK 17+ for Java/Kotlin compilation
