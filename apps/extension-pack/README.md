# Android DevKit Pack

A curated extension pack that sets up a complete Android development environment in VS Code with a single click. No Android Studio required.

## Included Extensions

| Extension | Purpose |
|-----------|---------|
| **[Android DevKit](https://marketplace.visualstudio.com/items?itemName=pavi2410.android-devkit)** | Device manager, Logcat viewer, wireless debugging, device file explorer, SDK auto-detection |
| **[Language Support for Java](https://marketplace.visualstudio.com/items?itemName=redhat.java)** | Full Java language support via Eclipse JDT LS — completion, navigation, refactoring |
| **[Gradle for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-gradle)** | Gradle build tool integration — run tasks, view dependencies, sync projects |
| **[Kotlin](https://marketplace.visualstudio.com/items?itemName=fwcd.kotlin)** | Kotlin language support — syntax highlighting, basic completion |
| **[XML](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-xml)** | XML editor for Android layouts, manifests, and resource files |

## Why Use This Pack?

- **One-click setup** — get a complete Android development environment instantly
- **Curated selection** — each extension is chosen for quality and Android compatibility
- **Lightweight** — only what you need, no bloat
- **Always up-to-date** — extensions update independently on their own release schedule

## Getting Started

1. Install this extension pack
2. Open an Android project folder (containing `build.gradle` or `AndroidManifest.xml`)
3. Android DevKit activates automatically — click its icon in the Activity Bar
4. Connect a device or start an emulator to begin

## Requirements

- **Android SDK** — install via Android Studio or the command-line tools; set `ANDROID_HOME` or let Android DevKit auto-detect it
- **JDK 17+** — required for Java/Kotlin compilation via the Language Support for Java extension
